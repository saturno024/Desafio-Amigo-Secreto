## 🚀 Instalación y Uso

### 📋 Prerrequisitos

- Navegador web moderno (Chrome 88+, Firefox 85+, Safari 14+, Edge 88+)
- ¡Eso es todo! Solo necesitas un navegador 🚀

### ⚡ Inicio Rápido

```bash
# Clonar el repositorio
git clone https://github.com/saturno024/Desafio-Amigo-Secreto-.git

# Navegar al directorio
cd Desafio-Amigo-Secreto-

# Abrir directamente en el navegador
start index.html
# O simplemente hacer doble clic en index.html
```

### 🔧 Configuración para Desarrollo (Opcional)

```bash
# Para desarrollo con Live Server en VSCode
code --install-extension ritwickdey.liveserver

# O usar cualquier servidor local simple
# (Solo si quieres ver cambios en tiempo real)
```

## 📖 Guía de Uso

### 1️⃣ **Agregar Participantes**
```
1. Escribir el nombre en el campo de entrada
2. Presionar Enter o hacer clic en "Adicionar"
3. El sistema validará automáticamente el nombre
4. Confirmación visual y sonora al agregar
```

### 2️⃣ **Realizar Sorteo**
```
1. Asegurar que hay al menos 2 participantes
2. Hacer clic en "Sortear amigo"
3. Ver el resultado con animación celebratoria
4. El ganador se muestra con efectos especiales
```

### 3️⃣ **Gestionar Lista**
```
1. Ver contador en tiempo real de participantes
2. Usar "Reiniciar" para limpiar todo
3. Confirmación antes de borrar datos
4. Estado visual del progreso
```
